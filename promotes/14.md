
1. 根据interface LocalFolder {
  id: string;
  name: string;
  path: string;
  songCount: number;
} 类型，在src-tauri/src/database 添加 local_folders Table
2. 在src-tauri/src/database/operations.rs 添加add_folder, get_folders (支持name 查询), delete_folders 函数并添加单元测试。
3. 并将这些api 添加到src-tauri/src/audio/commands.rs 中给前端调用。
4. 然后在src/hooks 添加这些函数调用和类型返回，并添加单元测试。最后src/components/Local/Local.tsx 调用这些函数。
5. 不要忘记运行cargo fmt -- --check，cargo clippy --all-features -- -D warnings -W clippy::pedantic -W clippy::nursery -W rust-2018-idioms，前端代码需要 code format 和 lint。
